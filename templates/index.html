Advertise with us

Mariana Tek's mobile app now allows advertising banners, so that you can reach loyal customers outside of the studio. Our platform will allow you to:

  • Insert an ad image and link it to a URL. For recommendations on the best size of your ad image, contact your account manager.
  • Serve different ads to different user segments. To see this in action, simply pass in an associated user ID with a segment you would like to target to the example ad here.

Create your own Advertising API

Have the ability to create your own campaigns by following the steps below:

  • Fork the demo repo here and modify server.js to create your own ad.
  • With the forked version, test your implementation of the advertising API with your ad content locally.
  • As part of your API, implement a response that returns the URL of the image, as well as the URL of the redirect link that you want to be shown for the requesting user.
  • Once the ad is ready to go live, communicate the URL for your API to your account manager to complete configuration.
